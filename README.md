### WebView 实现JS效果和a标签的点击事件

<p>目前很多android app都可以显示web页面的界面，嵌入式开发，这个界面一般都是WebView这个控件加载出来的，学习该控件可以为你的app开发提升扩展性。</p> 
<p>先说下WebView的一些优点：</p> 
<ol> 
 <li>可以直接显示和渲染web页面，直接显示网页</li> 
 <li>webview可以直接用html文件（网络上或本地assets中）作布局</li> 
 <li>和JavaScript交互调用</li> 
 <li>网页标签的点击事件</li> 
</ol>
